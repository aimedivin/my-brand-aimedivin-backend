# My Brand Backend API

## Project Overview
My Brand Backend API provide RESTfull endpoints which act as server-side for My brand project.

## Features
- Blog Management
- User authentication
- Handling contact messages

## Technologies Used
- Node.js
- Expresjs
- Mongodb
- Mongoose
- JWT (JSON Web Tokens) for authentication
